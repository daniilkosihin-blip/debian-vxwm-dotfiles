# 🌀 Debian vxwm & Picom Dotfiles

Автоматический установщик и конфигурационные файлы для создания легковесного, быстрого и красивого рабочего окружения на базе **Debian** с использованием модульного оконного менеджера **vxwm**.

---

## ✨ Основные особенности
* **vxwm ** — прокачанный и модульный форк `dwm` от *wh1tepearl* с бесконечных "рабочим столом" (infinite canvas) 
* **Picom** — современный композитный менеджер (сборка из исходников ветки *yshui*), обеспечивающий плавные анимации, тени и размытие (blur) окон.
* **Alacritty** — сверхбыстрый эмулятор терминала с аппаратным ускорением через графический процессор.


---

## 🚀 Быстрая установка (Установщик в 1 клик)

Скрипт автоматически обновит пакетную базу Debian, загрузит все необходимые библиотеки для сборки, скомпилирует `vxwm` и `picom` из исходного кода, применит кастомные конфигурации и запустит графическую сессию.

Для установки просто откройте терминал и выполните:

```bash
git clone https://github.com
cd debian-vxwm-dotfiles
chmod +x setup.sh
./setup.sh
```

> ⚠️ **Внимание:** Скрипт предназначен для чистой системы (минимальной установки Debian без тяжелых сред вроде GNOME/KDE). Перед запуском убедитесь, что ваши личные данные сохранены.(по поводу различных ошибок установки либо пишите в issues либо сами исправляйтя файл установщика) 

---

## ⚙️ Структура репозитория
* `setup.sh` — главный скрипт автоматического развертывания окружения.
* `vxwmconf/config.h` — кастомный конфигурационный файл для оконного менеджера (настроен под терминал Alacritty и запуск dmenu).
* `picom/` — конфигурация композитора (файлы настроек эффектов и анимаций).
* `.xinitrc` — сценарий инициализации графической сессии X-сервера **настройте его под ваш монитор (чтоб узнать подключенный монитор напишите в терминале xrandr потом замените в файле DP-0 на ваш подключенный интерфейс и уберите #** .
